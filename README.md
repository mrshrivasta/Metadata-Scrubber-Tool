## 🔎 About This Project

Files such as images, documents, and media often contain **hidden metadata** (EXIF data, author info, device details, timestamps, etc.) that can unintentionally expose sensitive information.

This tool helps users:

* Detect metadata inside files
* Understand what data is being shared
* Remove metadata before sharing files online

---

## ✨ Features

* 📂 Drag & drop file upload
* 🔍 Metadata extraction (EXIF and other formats)
* 🧹 One-click metadata removal
* 📁 Supports multiple file types (images, documents, media)
* 🔐 100% client-side processing (no uploads to server)
* ⚡ Fast and lightweight performance
* 📥 Download cleaned files instantly

---

## 🧠 Why Metadata Scrubbing Matters

Metadata can include:

* Device information
* GPS location
* Author or software details
* Hidden timestamps

Removing metadata helps:

* Protect personal privacy
* Prevent data leaks
* Maintain anonymity when sharing files

---

## ⚠️ IMPORTANT DISCLAIMER

This tool is designed for **privacy and educational purposes only**.

* It operates entirely in the browser and does **not upload files to any server**
* Metadata removal is performed using client-side techniques and may **not guarantee complete removal in all cases**
* Some advanced or deeply embedded metadata may remain depending on file format and browser limitations

---

## ⚠️ LIMITATIONS

Due to browser constraints:

* Large files may impact performance
* Some file formats may have partial metadata removal
* Video/audio metadata removal is limited
* Deep forensic-level metadata removal is not supported
